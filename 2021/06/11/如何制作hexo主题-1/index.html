前言

本教程为博主折腾过程的总结。

准备工作

首先，需要一个网页的模板，可以在html5up上获取，这里我选择Photon。

官网文档

来到hexo官网，官网文档是必需要看的。

来到主题页面我们可以看到主题的基本结构。

请务必认真阅读。

本地文件

如果你已经看完上述文档就可以开始制作了，首先需要一个博客，如hexo文档所示，这里就不过多赘述。
成功运行一次hexo后使用VScode(或者其他软件)打开博客目录，可以看见一个空的themes目录(一般情况是空的)。

这时候就需要文档里面的结构目录了。

给主题新建一个目录，名称随意(这里我取名为start)。

在博客主目录下的_config.yml中修改theme为主题目录名称(注意是博客目录不是主题目录)。

模板引擎

主题模板需要一个引擎才能被博客使用，这里我使用ejs，使用cmd转到博客目录,输入npm install hexo-renderer-ejs --save或者cnpm install hexo-renderer-ejs --save指令来安装ejs。
